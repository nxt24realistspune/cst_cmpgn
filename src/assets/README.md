# Assets Folder

## Logo File

Please place your **MPLogo.png** file in this directory.

### Required File:
- **Filename:** `MPLogo.png`
- **Location:** `/src/assets/MPLogo.png`
- **Recommended Size:** 128x128px or larger (will be displayed at 64x64px)
- **Format:** PNG with transparent background (recommended)

### Current Usage:
The logo is used in the QA Platform header component at:
- Component: `src/app/components/qa-platform/qa-platform.component.html`
- Referenced as: `assets/MPLogo.png`

Once you add the logo file here, it will automatically appear in the application header.
